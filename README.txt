
VÆLINE – Nasadenie na Render.com (Stripe Checkout)

1. Nahraj tento projekt na GitHub (nový repozitár, napr. 'vaeline-store')
2. Choď na https://render.com → New → Web Service
3. Vyber GitHub ako zdroj → vyber repozitár s týmto projektom
4. Render načíta konfiguráciu z 'render.yaml'
5. Hotovo – Render ti nasadí stránku s backendom + Stripe

Platba bude testovacia – 20€ cez Stripe Checkout.
